# 🌾 Village Connect - A Smart Village Directory App

## 📌 Overview

Village Connect is a smart village directory and assistance application designed to help residents easily access essential contacts, services, and announcements within their village. The platform aims to bridge the communication gap between villagers and local authorities using a simple and user-friendly interface.

---

## 🚀 Features

* 📇 **Village Directory** – Categorized contact list (officials, services, etc.)
* 🚨 **Emergency Contacts** – Quick access to important emergency numbers
* 🏪 **Local Services** – Information about nearby services like shops, hospitals, etc.
* 📢 **Panchayat Announcements** – Stay updated with local news and updates
* 📝 **Feedback & Complaints** – Users can submit issues and suggestions
* 📱 **Responsive UI** – Clean and intuitive design for better user experience

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/VyshnaviTeeguri/Village-Connect.git
```

2. Navigate to the project folder:

```
cd Village-Connect
```

3. Install dependencies:

```
flutter pub get
```

4. Run the app:

```
flutter run
```
